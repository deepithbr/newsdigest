# News Brief

Daily personal news briefing generated by GitHub Actions and published with GitHub Pages.

## How it works

- GitHub Actions runs daily at `03:30 UTC`, which is `09:00 IST`.
- The workflow runs `scripts/daily_brief.py`.
- The latest HTML is published as `public/index.html`.
- Dated archives are written under `public/archive/`.

## Manual run

```bash
python scripts/daily_brief.py --hours 36
```

Open:

```text
public/index.html
```

## Sources

Edit `config/sources.json` to change sources. GDELT sources are included but disabled by default because they can be rate-limited.

## GitHub Pages setup

After pushing this repository to GitHub:

1. Open repository settings.
2. Go to `Pages`.
3. Under `Build and deployment`, set `Source` to `GitHub Actions`.
4. Run the `Daily News Brief` workflow manually once from the `Actions` tab.

The stable reading URL will be the repository's GitHub Pages URL.
